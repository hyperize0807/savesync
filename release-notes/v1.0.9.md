## ⚡ 폴더 동기화 속도 개선 (병렬 전송)

파일이 많은 폴더를 동기화할 때 파일당 네트워크 왕복(2~3초)이 쌓여 수 분이 걸리던 문제를 개선했습니다.

- 여러 파일을 **동시에 전송**하도록 변경(기본 8개 병렬) → 파일이 많을수록 크게 단축(대략 **7~8배**).
- 파일별 충돌 해결·백업·부분 실패 격리 등 기존 안전장치는 **그대로 유지**합니다.

## 🛟 로컬 폴더 미지정 프로필 안전장치

프로필을 드라이브에서 가져온 뒤 **로컬 세이브 폴더를 지정하지 않은 상태**에서 의도치 않은 폴더가 동기화되던 위험을 막았습니다.

- 로컬 폴더가 비어 있는 프로필은 동기화에서 **건너뜁니다**(수동 '지금 동기화' 시에는 경고 팝업).
- 프로필 목록에서 해당 프로필에 **`⚠ 폴더 미지정`** 표시, 가져오기 완료 안내에도 설정이 필요한 프로필을 명시.

## 🧭 설정 창 UI 개선

- **'현재 입력을 프로필에 적용' 버튼 제거** — 하단 **저장**이 이미 편집 내용을 반영하므로 중복이었습니다. 이제 편집 후 **저장**만 누르면 됩니다.
- **미저장 변경 확인**: 저장하지 않고 다른 프로필로 전환하거나, 다른 탭으로 이동하거나, 창을 닫으려 하면 확인 팝업이 뜹니다.
  - **저장** → 저장하고 하려던 동작 수행
  - **아니오** → 저장하지 않고 수행(변경 폐기)
  - **취소** → 아무것도 하지 않고 현재 화면 유지

> 설정·인증 토큰·백업은 `%APPDATA%\SaveSync` 에 보관되며, 업데이트로 exe 를 교체해도 그대로 유지됩니다.
